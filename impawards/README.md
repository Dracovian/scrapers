# wallhaven-scraper

Downloads all available movie posters.

## Prerequisites
* python3
* bs4

## Usage
```
python3 impawards-scraper.py 
```

Sample output
```
memes@pepe:~/git/scrapers/wallhaven-scraper$  python3 impawards-scraper.py 
==== Page 1  ====
Aardvark (2018)
Aaron's Blood (2017)
Aaron's Blood (2017)
Aaron Loves Angela (1975)
Aatsinki: The Story of Arctic Cowboys (2014)

```
